CommentIQ

Privacy Policy

Last updated: 28 June 2026

CommentIQ is built to be private by design. We do not collect, transmit, sell, or share your data — no analytics, no tracking, no servers, no accounts. Everything the extension does runs locally in your browser using Chrome's Gemini Nano.

The short version

  • No backend servers. CommentIQ has no infrastructure — no data leaves your device.
  • No personal data collection. We don't see your YouTube username, video history, comment list, or analysis results.
  • No analytics or tracking. No event logging, no user tracking, no telemetry.
  • No accounts required. Install and use immediately — no sign-up needed.
  • 100% on-device analysis. Comments are fetched locally, analysed by Gemini Nano on your machine, stored only in your browser's storage.

What CommentIQ accesses

To fetch and analyse comments, the extension reads:

  • The current YouTube video ID — to fetch that video's comments via the Innertube API
  • Comment text — fetched from YouTube's public comment API, used only for local analysis
  • Comment metadata (optional) — like count, publish date, author (if extracted from Innertube, used only for local sorting/filtering)

Comments are processed entirely in your browser using Gemini Nano. No comments or analysis results are sent anywhere.

What is stored, and where

CommentIQ stores the following locally on your device only, using Chrome's storage.local API:

  • Your theme preference (Light, Dark, Midnight, Cream)
  • Analysis results for the current video (cached so re-opening the video reuses results)
  • Daily quota usage (free plan limit tracking)
  • Pro subscription status (if purchased)

This data never leaves your browser. It is not sent to any server, and it is not accessible to us or any third party.

Network requests CommentIQ makes

  • YouTube Innertube API: Fetches comment data from YouTube's public API when you click "Analyse". This request includes your video ID and nothing else about you.
  • No other requests: CommentIQ makes no requests to any other service. No analytics, no crash reporting, no telemetry.

Optional features (Pro)

When you upgrade to Pro, CommentIQ stores additional data locally:

  • Analysis history (up to 90 days of past video analyses)
  • Your Pro subscription key (used only for local validation, never transmitted)

This data also stays on your device and is never sent to our servers.

Gemini Nano (Chrome's built-in AI)

CommentIQ uses Chrome's Prompt API to access Gemini Nano, Google's on-device language model. Gemini Nano runs locally on your machine — the model operates entirely within Chrome's sandbox and never makes requests to Google's servers during analysis. See Google's Chrome documentation for details on Gemini Nano's privacy model.

Changes to this policy

If this policy changes, the "Last updated" date above will be revised. We'll notify early-access testers of material changes.
